# Ctrl-Alt.Ninja
Become a keyboard ninja.

Play now at [https://ctrl-alt.ninja/](https://ctrl-alt.ninja/).

## Development
```sh
npm install
gulp build
gulp serve
```

Open http://localhost:3000/ in your browser. Running `gulp build` again will cause the page to automatically reload.
